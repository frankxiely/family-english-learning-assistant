# 内容知识库 v1

这个目录用于沉淀“每天 lesson JSON 怎么写”和“长期学习路线图怎么排”的依据。

它不是用户学习记录库，也不是外部教材搬运库。这里保存的是：

- 可引用的教学标准和资料来源。
- 我们自己的课程分层规则。
- 每天学习量规则。
- 可机器读取的路线图 JSON。
- 课文、单词、讲解、测试和音频资产的生成约束。

## 分层

1. 标准层：用 CEFR、Cambridge Pre-A1/A1、Oxford 3000 等资料确定大致难度。
2. 知识层：记录音标、字母、音节、拼读、词性、句型、语法点。
3. 场景层：记录问候、自我介绍、家庭、购物、银行接待、会议表达等使用场景。
4. 路线层：把知识点和场景模块排成可执行的每日路线。
5. lesson 层：把某一天的路线项转译成 lesson JSON。
6. 资产层：记录图片、音频、头像等素材的来源、生成参数和本地路径。

## v1.1 原则

- 首位学习者是接近 0 基础的成人，不直接套儿童教材语气。
- 可以参考儿童 Pre-A1 的小步长和高频词，但课文、例句和场景要成人化。
- 每天默认 30 分钟，优先稳定完成。
- 每天只推进 1 个主知识点，配 3 到 4 个新词、1 篇短对话、3 到 5 道测试题。
- 路线图决定“今天学什么”，lesson JSON 决定“今天怎么呈现”。
- 外部资料只作为分级和结构参考，不直接复制教材正文、音频、图片或整套题目。

## 文件

- `sources_and_materials.md`：资料来源、使用方式和版权边界。
- `route_design_v1.md`：长期路线图和每日学习量规则。
- `lesson_json_authoring_rules.md`：lesson JSON 写作约束。
- `audio_strategy_v1.md`：自然音频和课文朗读方案。
- `../../data/teaching_knowledge/source_catalog.v1.json`：资料来源的机器可读目录。
- `../../data/teaching_knowledge/starter_phonics_route.v1.json`：零基础首阶段路线图初稿。
