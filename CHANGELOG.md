# Changelog

## v1.1.0-local-web

- 初始化项目骨架。
- 新增 SQLite schema 和种子数据。
- 新增本地 template/manual/mock provider 规划。
- 新增标准 lesson JSON 入库链路。
- 新增学习复盘 JSON 资产规划。
- 新增模块 debug 脚本。
- 新增管理员后台闭环：生成草稿、编辑 lesson JSON、保存备注、发布、回滚、修改用户画像。
- 新增学习端闭环：单词发音、三态掌握、课文播放、测试选择、自评、难点记录和完成提交。
- 新增真实学习记录写入：`daily_progress`、`quiz_attempts`、`question_answers`、`error_records`、`difficulty_records`、`vocabulary_mastery` 和 `review_queue`。
- 新增每日复盘和 7 天简版进度分析，并写入 RAG 来源表。
- 新增 Markdown/CSV 学习记录导出脚本和 API。
- 修复初始化种子数据覆盖学习状态的问题。
- 修复发布课程时 `REPLACE` 破坏历史答题外键的问题，改用 upsert。
- 新增本地登录接口和预置学习者账号 `ViZhang`。
- 区分唯一登录账号和可重复用户名；当前学习者登录账号为 `ViZhang`，用户名为 `Vi`。
- 新增本地管理员账号 `admin_xly`，`/admin` 必须管理员登录，管理 API 和 debug API 必须携带管理员 token。
- 新增测试管理员账号 `AdminXLY`，展示名为 `Admin_1`，并接入管理员头像。
- `AdminXLY` 同时绑定学习者身份 `user_admin_1`，可进入学习端；学习者画像为 25 岁、硕士研究生、雅思 7.0 水平。
- 重做管理员工作台：用户目录、单用户工作台、未发布 lesson 草稿 JSON、备注、AI/模板调整、人工编辑、连续还原、管理员反馈日志和学习画像概览。
- 新增 `lesson_draft_workspaces` 和 `lesson_draft_snapshots`，未发布课程不再进入正式 `lesson_json_assets`。
- 新增课后复盘后自动生成下一天未发布草稿的本地流程；当天课程由草稿发布后再进入学习端。
- 接入 `antd` 和 `antd-mobile`；学习端改为路径式学习体验，管理端改为成熟后台工作台布局，减少原先的卡片堆叠感。
- 将学习端 `/learn`、登录页 `/login`、管理员端 `/admin` 分离。
- 学习端登录页新增低强调“管理员登录”小按钮，主路径仍服务学习者登录。
- 登录页 `Vi` 圆形标识替换为线条风 Logo 占位稿：打开的书页、对话气泡、抽象 `M` 和右上角星标。
- 管理员登录页默认账号改为 `AdminXLY`。
- 移除学习者端的后台式进度页面。
- 新增 `/api/learning/route-map`，学习端可读取长期课程路线图，路线图负责规定每日主题。
- 学习端从长页面改为首页、学习进度、逐步学习、我的四个交互视图；今日课程用单步播放器递进完成。
- 我的页面新增本地账号设置：修改用户名、修改密码。
- 首页和我的页面接入用户指定头像图片。
- 单词页新增配图、进入自动发音、“知道 / 模糊 / 不知道”点击即继续。
- 新增本地单词图片资产生成与 `asset_sources` 记录。
- 新增 `vocabulary_mastery_events`，单词掌握选择保留事件历史，同时更新当前状态和复习队列。
- 课文页新增正常/慢速播放、逐句播放、今日单词高亮和难词释义弹窗。
- 课文页改为“原文页 + 译文页”：原文页保留角色头像和右侧小播放按钮，译文页集中展示难词词义和中文译文。
- 学习完成后新增奖杯完成页；lesson JSON 新增完成鼓励语字段。
- lesson JSON 新增首页鼓励语字段；学习首页改为显示中英文鼓励语和累计已学单词，移除时长、保存提示和装饰元素。
- 我的页面账号设置改为按钮入口，修改用户名和修改密码表单默认折叠。
- 放大学习端手机阅读字体层级；单词页取消默认“模糊”选中，词性改为中文，词义放大，并移除单词步骤的“继续”按钮。
- 学习首页新增今日任务块；学习进度页新增课程详情面板，支持查看已完成复盘、今日继续和未来预告。
- 我的页面改为左上角设置入口，账号设置拆为独立页面，并在我的页补充学习档案区域。
- 课文原文和译文合并为同一个主模块，译文改为课文模块内部子页；逐句播放改为喇叭图标按钮。
- 学习播放器底部按钮改为“上一模块 / 下一模块”；课文和知识讲解新增卡片内分页，不再依赖拖拽阅读。
- 测试交互改为答对自动下一题、答错停留并显示解析；前端提交 `quiz_mistakes`，后端写入错题记录。
- lesson JSON 新增 `progress_summary`，进度页展示学习路径模块、主要知识点和课文模块。
- 我的页统计改为累计完成测试题、学习天数、完成模块和当前模块；累计完成测试题按用户和题目主键去重统计。
- 路线图接口新增每一天的摘要字段，未来课程可展示学习模块、主要知识点和课文/场景模块，不需要提前生成完整 lesson JSON。
- 课文和知识讲解卡片分页新增“上一页 / 下一页”；知识讲解标题改为当天学习主题。
- 学习端新增按钮交互体系：主行动下沉回弹、模块导航方向反馈、单词三态选择反馈、测试错误摇动、音频播放波纹和原文/译文分段滑块。
- 新增学习进度记忆：从学习播放器返回主页后，再次点击继续会回到退出时的位置；完成后按钮改为复习当天内容。
- 我的页面移除今日复盘长文本和继续复习按钮，改为大头像横向身份卡，鼓励语来自当前阶段。
- 新增对话角色头像素材库和头像风格 prompt 文档，支持 teacher、doctor、mom 等角色头像抽取。
- 头像素材库新增 `admin-male.png`，可用于管理员测试账号和课文中的 administrator/admin 角色。
- 新增 `scripts/reset_user_mom_learning_records.sql`，可清理首位学习者的学习记录并保留账号、画像、路线图和教学知识库。
- 新增内容知识库第一版：资料来源、使用边界、路线图设计、lesson JSON 写作规则和音频策略。
- 新增机器可读知识库文件：`data/teaching_knowledge/source_catalog.v1.json` 和 `data/teaching_knowledge/starter_phonics_route.v1.json`，用于约束后续 lesson JSON 与长期路线图生成。
- 后端 lesson 生成器接入 `starter_phonics_route.v1.json`，按学习天数选择内容路线项，并写入 `route_basis`、`progress_summary`、`source_basis` 和 `audio_assets`。
- 新增后端 pytest，覆盖内容知识库路线生成、lesson JSON 标准化、发布和学习提交。
- 新增 Playwright 手机端 E2E，覆盖登录、今日学习、单词掌握、课文、知识讲解、测试、完成和进入“我的”页面。
- 新增 `test:api` 和 `test:e2e` 脚本，并安装 `@playwright/test`。
- 将当前课程重构为 30 分钟“音标第一课：/iː/ 和 /ɪ/”，包含教材式对话和系统知识讲解。
- 新增 `teaching_knowledge_assets` 教学知识库表和首个音标知识资产。
- 将前端主色调调整为舒适蓝绿风格。
